# Animal Farm

A farm with animals that go on amazing dining and sleeping adventures together.

## Getting Started

```
git clone https://github.com/inkhsutesou/animal-farm.git
npm install
npm run start
```

http://localhost:9000/

YOU DID IT !!

## Running the tests

```
npm run test
```
